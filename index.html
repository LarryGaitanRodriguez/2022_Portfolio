Mobile Resturant Visual Menu Page

Being a mobile first responsive menu page, this site was a proof of concept for an updated visual menu page for a resturant. Visual in this context means the site displays the physical printed menu. Assets were downloaded off the current website and adapted to a more sleek design thats intended to make navigation easier for the customer. The website is made with HTML and Bootstrap.

Mobile Resturant Text Menu Page

Being a mobile first responsive menu page, this site was a proof of concept for an updated text menu page for a resturant. Text in this context means the menu is typed in directly into the website. It isn't represented by an image of the physical printed menu. This allows for a quicker update menu items, descriptions, images, and prices.

Javascript exercise - Mock Webdev company

Part of a website development college course assignment, this mock web dev company page was created in order to practice HTML, CSS, Javascript, and JQuery. The main features are the image carousel, the full page hamburger menu, and a quirky call function that pulls weather data from the National Weather Service API.

PHP and Sessions exercise - Login page

Part of a website development college course assignment, this login page was create to practice PHP and the concept of user sessions. This site used Bootstrap for the front-end.
